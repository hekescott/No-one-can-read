### _2-我也要做一个论坛

```php
// 1.我不怎么喜欢论坛，是因为那不是我想要的论坛

// 2.Now，现在我要做一个，我希望的论坛了，先本地上线一把。

// 3.什么是人脑记忆模式，通过判断，把没有强化过的记忆舍弃，排除一些杂乱因素。

// 4.应用在论坛上就是，删除那些无效信息，在无人回应后的第5天，或者第10天，当然也不能这样判断，回应的人数越多，有效期越长。当人数超过100人，或者其他数字，永久有效期。当人数小于5天且10天无应答，那么就判断为死贴，予以删除。

// 5.论坛删除模式，帖子可以分为，有趣，无聊，如果点无聊的人多了，或者无回应状态，那么也判断为死贴。

// 6.兴趣点集中，把同样类型的帖子进行排比竞争，自动排除末位，同样推荐前3帖子，聚集精华，形成规模优势。

// 7.回帖，要像聊天室一样，而不是一个刻板的论坛，当然最终的呈现可能偏论坛一些，但是他的样子，表现形式应该是聊天室（群聊）的手法。

// 8.论坛：永久聊天室，同样为了需要也可以创建临时聊天室，聊天记录，以最后一个聊天记录过24h，删除。或者创建人主动解散，需最活跃的前几名成员同意，如果12小时内无应答，则默认同意。

// 9.进入论坛，小秘书，小机器人接待（智能search），记录你在论坛的行为，给出建议，等等

// 10.用户反馈系统，一个以人为本的论坛，肯定要尊重各个人的意见，收集整理意见。对于突出问题，进行讨论和改进。

// 11.以前的论坛，分坛都是一个一个单独的房间一样，非常死板。现在论坛要把分论坛设计成标签模式，一个主题可以拥有N个标签，用户可以通过标签进入，也可以直接通过主题进入。
```
